The list provided here contains data on the quests that are available to be
undertaken in Dragon Quest IX.  The list goes in order of quest number, which
is rarely the order in which you actually encounter the quest in-game.

The list contains the following data:

 Quest #: This is the number of the quest related to the in-game sequence.
          Next to the quest number is the quest name.
Location: The city in which you will find the quest starting place, and the
          specific NPC or object that you receive the quest from.
 Request: The task required to complete the quest.
Solution: Advice, suggestions and help on how to complete the various quests.
  Reward: Reward that is received for completing the quest.
 Pre-req: If the quest has any pre-requisites that need to be met before you
          are able to receive this quest.
  Repeat: Yes means that this quest is repeatable, and no means that it is
          not repeatable.
   Notes: Any additional notes regarding the quest.
